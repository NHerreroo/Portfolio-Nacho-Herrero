Proyectos

Froguenture

Froguventure es un videojuego 3D desarrollado con Godot que utiliza generación procedural de niveles mediante el algoritmo Drunkard Walk. El juego crea mapas únicos en cada partida para ofrecer variedad y desafío.

Combina mecánicas roguelike con progresión de personaje, permitiendo al jugador mejorar habilidades y superar obstáculos. El diseño se centra en la exploración y la rejugabilidad, usando scripts en GDScript para controlar la lógica del juego y la generación dinámica del mundo.

Pallaresa File Manager

Este proyecto fue un encargo para la Escuela Pallaresa. Se trata de una plataforma web que se asemeja bastante a Google Drive, diseñada para gestionar archivos de forma rápida y sencilla. En ella, los usuarios pueden tener roles asignados, con permisos específicos que determinan quién puede acceder, subir o descargar los archivos. La aplicación está desarrollada en Java, utilizando Spring Boot como framework backend, PostgreSQL como sistema de gestión de bases de datos y React para el frontend.

Sobre Mi

Me llamo Nacho Herrero y soy desarrollador de software. Empecé en la programación hace dos años, combinando mis estudios con proyectos personales. Actualmente trabajo en desarrollos propios y colaboro con otros compañeros para crear aplicaciones y juegos desde cero.

Además, he liderado varios equipos de desarrollo en diferentes proyectos, entre ellos una plataforma web estilo Google Drive, donde coordiné tanto la parte técnica como la organización del equipo para cumplir objetivos en plazos ajustados.
Soy una persona proactiva, con iniciativa y con facilidad para trabajar en equipo, cualidades que me han permitido asumir roles de liderazgo desde etapas tempranas. Siempre estoy en busca de nuevos retos que me permitan crecer profesionalmente. En mi tiempo libre disfruto descubriendo nuevas tecnologías y tocando la guitarra como parte de un grupo musical.
